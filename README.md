Scientific work  
===============  
  
1. Prediction of Zero Field Splittings (ZFS) in metal complex FeNO  
from ab initio using unresctricted Density Functional Theory (DFT).  
I implemented in C a program that takes converged molecular orbitals (MOs)  
from Gaussian software and I do post-DFT calculations to compute ZFS.  
It required to learn how to implement molecular integrals (MO integrals)  
such as two-electron MO integral, spin-orbit coupling integrals.  
Title of Paper: "Accurate Calculatio of Zero-Field Splittings of (Bio)inorganic  
Complexes:  Application to an {FeNO}^7 (S = 3/2) Compound.  
  
File: Aquino2009_FENOpaper.pdf  
  
2. Computation of Electric Field Gradients (EFG) for relativistic systems  
using DFT.  
I implemented a Fortran module within HPC quantum chemistry software NWCHem  
that computes EFG for relativistic systems using Zero-Order Regular Approximation  
(ZORA).  
Title of Paper: "Electric Field Gradients Calculated from Two-Component  
Hybrid Density Functional Theory Including Spin-Orbit Coupling"  
  
File: Aquino2011_EFG-Z4.pdf  
  
3. Computation of Nuclear Magnetic Shielding and g-Shifts using ZORA for relativistic  
systems.  
I implemented a Fortran module within HPC quantum chemistry software NWChem  
that computes NMR and g-shifts for relativistic systems using ZORA.  
Title: "Scalar Relativistic Computations of Nuclear Magnetic Shielding and g-Shifts  
with the Zeroth-Order Regular Approximation and Range-Separated Hybrid Density Functionals"  
  
File: Aquino2011_NMRCS_g-shift_zora.pdf
